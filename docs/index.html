react-highlighter-ts

This is a TypeScript rewrite of the popular npm package react-highligher, which has no type support, has very out of date dependencies, and seems to be abandoned (no activity since 2018).

See generated type documentation for details

Contributing

See something amiss? Please open a PR.

Development setup

This library does not use any global dependencies, so a simple npm install should suffice.

Testing

Tests have been adapted from the original react-highlighter codebase. Tests use Jest + Enzyme. 

npm run test

or to watch file changes: 

npm run test:watch
